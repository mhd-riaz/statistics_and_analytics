# Data

Data is a collected facts, measurements or observation such as number, words, images, ...

## Attributes

Attributes are the characteristics or properties of the data that can be used to describe, analyze, and understand the data. They provide information about the data and help in categorizing, organizing, and interpreting it. 

Attributes can be of various types, such as:
1. **Numerical Attributes**: These attributes represent ***quantitative data*** and can be measured on a numerical scale. Examples include age, height, weight, and income.
2. **Categorical Attributes**: These attributes represent ***qualitative data*** and can be divided into distinct categories or groups. Examples


## Data sources

| **Primary data**                                                                 | **Secondary data**                                                          |
| :------------------------------------------------------------------------------- | :-------------------------------------------------------------------------- |
| the data that is collected continuously                                          | type of data that is already been collected, stored as archive or published |
| is fresh, collected for specific analysis, ingested by a computer during runtime | is a bit old, as it was collected and stored already                        |

During old days, the rule of thumb was to use secondary data as collection of primary data was not feasible.
But due to the advancement and automation in place, its now more easy to collect realtime(primary) data

## Types of data

1. Qualitative data (categorical data)
   - are non-numerical in nature
   - e.g.: gender, religion, place of birth
2. Quantitative data (numerical data)
   - are numerical in nature
   - e.g.: balance in bank, total members in your family


### Types of data sets

1. Records:
   1. Relational records
   2. Data matrix: e.g [numerical matrix, cross tabs]
   3. Document data: text document
   4. Transactional data
2. Graph and network:
   1. World Wide Web
   2. Social / information networks
   3. Molecular structures
3. Ordered:
   1. Video data: sequence of images
   2. Temporal data: time-series
   3. Sequential data: transaction sequences
   4. Genetic sequence data
4. Spatial, image and multimedia:
   1. Spatial data: maps
   2. image data
   3. video data

## Qualitative data types:

1. Nominal
   - categorical data
   - e.g. : HairColour [black, grey, white, blond], marital status [married, single, divorced], id number, zip code
   - NOTE: though they have number, arithmetical operation will make no sense here, eg. squaring of zip code or adding two ids; such data is called nominal
2. Binary
   - this is nominal data, but has only two outcomes (yes/no, 0/1, True/False)
   - Symmetric binary
     - Both outcomes are equally important
     - e.g: gender( male, female)
   - Asymmetric binary
     - Outcomes are not equally important
     - e.g: medical test result (positive or negative)
     - NOTE: the best practice is to assign true value to the important outcome (e.g. HIV positive)
3. Ordinal
   - Values with meaningful order (ranking)
   - NOTE here you cannot measure the value between the ranks
   - e.g.: {small, medium, large}, army ranking, grading system

## Quantitative data types:

1. Quantity 
   - integer (1,2,3,4, ...) or real number (175 cm, 175.6 cm, 175.65 cm)
   - Interval
     - Measured on a scale of equal-sized units
     - values have order (e.g.: temperature, calendar dates)
     - No true zero-point (e.g.: the birth of a individual is not on year 0, but age is calculated from the day it is born, thus the imaginary 0)
   - Ratio
       - is a measurement that is done relative to the unit of measurement
